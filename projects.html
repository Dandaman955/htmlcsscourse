Game Genie Address Decoder

This was the first homebrew program that I had written from scratch, and its purpose was to decode the Game Genie format into an address:patch format, as the Game Genie format is convoluted and involves many bit shifts across several byte boundaries. LINK

ISO9660 SCD

This is a program which generates a readable disk image to the ISO 9660 standards, created to allow for more flexibility with input data, designed for Sega CD games to follow the format listed in the technical specifications that the other ISO file system programs seem to struggle with. Mega Drive Input Test

This was a controller diagnostic ROM I had created for the Mega Drive designed to test controller inputs over a 2 second window. Since these consoles are getting older and the internals are wearing down, it's tailor made to test actual hardware with a flashcart. LINK

Sega Master System Checksum Fixer:

This was a project I had created to fix the checksum that the Master System would check to allow homebrew games to run, as the bootstrap ROM's code wouldn't allow the game to run if the checksum was incorrect. I later found out that flashcarts wouldn't check the checksum somehow, however I still feel that it was a good exercise in translating Z80 code to C, especially for the checksum calculation which uses an 'add with carry' instruction which takes advantage of the Z80 processor's internal carry flag.
